# WarrantConnect

Landing page futurista para `warrantconnect.com`, criada com Next.js, React, Tailwind CSS e Framer Motion.

## Como rodar

```bash
npm install
npm run dev
```

## Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Lucide React

## Estrutura

```text
app/
components/
sections/
styles/
```

## Destaques

- Hero com visual tecnológico, grid e partículas animadas
- Cards de serviços para os subdomínios da plataforma
- Seções de infraestrutura, arquitetura e status
- Tema dark/light com estética futurista
- SEO básico com metadata e OpenGraph
